# 🧠 Autism Spectrum Disorder Prediction System

نظام متقدم للتنبؤ باضطراب طيف التوحد باستخدام تعلم الآلة

## 📋 المحتويات

- `Autism_Prediction_Using_ML.ipynb` - دفتر Jupyter مع تدريب النموذج
- `app.py` - واجهة Streamlit التفاعلية
- `random_forest_model.pkl` - النموذج المحفوظ
- `Autism_data.csv` - بيانات التدريب
- `requirements.txt` - المكتبات المطلوبة

## 🚀 البدء السريع

### الخطوة 1: تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### الخطوة 2: تشغيل التطبيق
```bash
streamlit run app.py
```

سيفتح المتصفح تلقائياً على الرابط:
```
http://localhost:8501
```

## 🎯 المميزات

### 📊 علامة التنبؤ الفردية
- إدخال معلومات شخصية (العمر، الجنس، الجنسية)
- الإجابة على 10 أسئلة فحص التوحد
- الحصول على نسبة المخاطر فوراً
- رسم بياني يوضح مستوى المخاطر

### 📈 الاختبار الجماعي

- عرض النتائج في جدول منسق
- إحصائيات موجزة

- أداء النموذج
- توثيق كامل

- **F1 Score**: عالي
- **ROC-AUC**: ممتاز

- التركيز على التفاصيل
- التبديل بين المهام

- الانزعاج من التغييرات
- الاهتمامات الخاصة
- ملاحظة الأنماط
- تفضيل الروتين
